fastCRW Crawl URL API

Hosted: https://api.fastcrw.com · Self-hosted: http://localhost:3000

Auth (hosted only): Authorization: Bearer $CRW_API_KEY

Self-hosted: no auth required by default.

POST /v1/crawl

Start a multi-page crawl from a seed URL. Returns a job ID; poll for status.

Request body:

{
  "url": "https://example.com",
  "limit": 50,
  "scrapeOptions": { "formats": ["markdown"] }
}

curl:

curl -X POST https://api.fastcrw.com/v1/crawl \
  -H "Authorization: Bearer $CRW_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"url":"https://example.com","limit":50,"scrapeOptions":{"formats":["markdown"]}}'

Response shape: {"success": true, "id": "...", "url": ".../v1/crawl/{id}"}

GET /v1/crawl/{id}

Poll for crawl progress and results.

curl https://api.fastcrw.com/v1/crawl/$JOB_ID \
  -H "Authorization: Bearer $CRW_API_KEY"

Python (stdlib):

#!/usr/bin/env python3
import json, os, time, urllib.request

def post(url, body):
    req = urllib.request.Request(
        url,
        data=json.dumps(body).encode(),
        headers={"Content-Type": "application/json",
                 "Authorization": f"Bearer {os.environ['CRW_API_KEY']}"},
        method="POST",
    )
    with urllib.request.urlopen(req) as r:
        return json.loads(r.read())

def get(url):
    req = urllib.request.Request(
        url, headers={"Authorization": f"Bearer {os.environ['CRW_API_KEY']}"}
    )
    with urllib.request.urlopen(req) as r:
        return json.loads(r.read())

start = post("https://api.fastcrw.com/v1/crawl",
             {"url": "https://example.com", "limit": 50,
              "scrapeOptions": {"formats": ["markdown"]}})
job_id = start["id"]

while True:
    s = get(f"https://api.fastcrw.com/v1/crawl/{job_id}")
    if s.get("status") in ("completed", "failed"):
        break
    time.sleep(2)
print(s["data"][0]["markdown"][:500])
# Run with: python3 crawl.py
